Event details

The immersion day will be practical in nature and will require the following:

  • A laptop with Wi-Fi running Microsoft Windows, Mac OS X, or Linux.
  • An Internet browser such as Chrome, Firefox, Safari, or Edge.
  • Internet access and port 22 access to the internet.
  • Basic hands on experience in databases, operating systems and applications

Lunch and catering will be provided throughout the day.

  • 14th June

09:00 - 10:00 Introductions to AWS Migrations

In this session, you will learn strategies for planning and migrating existing workloads to the AWS Cloud, including best practices regarding landing zone deployments, as well as network connectivity.

10:00 - 12:00 Amazon EC2 and server migration

In this session, we will look at how rehosting your applications can be automated with tools such as AWS Server Migration Service and CloudEndure, to accelerate your migration to the cloud.

12:00 - 13:00 Lunch

Lunch will be provided by Amazon Web Services.

13:00 - 15:00 Data Migration

In this session, we will look at different methods to migrate data layers to the cloud, such as propriatry and open-source databases such as Microsoft SQL and MySQL respectively.

15:00 - 17:00 Legacy application Migration to the cloud

In this session, we will be looking at how organisations can simplify legacy application deployments and End User Computing by using application virtualisation capabilities, such as Amazon Workspaces and Amazon AppStream.
