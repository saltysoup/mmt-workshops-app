About the event

Modernising applications using docker containers is a fast growing trend for organisations to help remove technical debt on their journey into longer term goals such as a microservices architecture.

In this invitation only one day workshop, we will explore the container ecosystem on AWS and attendees will learn about how running workloads on containers can unlock critical business values, as well as gaining hands-on lab experience by deploying applications on the Amazon ECS platform.

  • Learning Objectives
  • Target Audience

  • Business values of Modern Application Development
  • Introduction to Docker and Containers ecosystem on AWS
  • Monolith to Microservices patterns
  • Tips and Tricks on running Microservices
  • Hands on lab for all of the above

The workshop is divided into 2 halves;

  • A high level overview of business outcomes through Modern Application Developement
  • A hands on lab on migrating monolith to microservices on Amazon ECS containers


Who should attend?

  • Solution architects
  • System and software developers
  • Development managers
  • IT professionals
  • IT Leaders

Event details

The workshop will be practical in nature and will require the following:

  • A laptop with Wi-Fi running Microsoft Windows, Mac OS X, or Linux.
  • An Internet browser such as Chrome, Firefox, Safari, or Edge.
  • Internet access and port 22 access to the internet.
  • Basic hands on experience in databases, operating systems and applications
