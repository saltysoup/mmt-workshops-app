About the event

Come join AWS to learn how to build a modern data capability to enable data driven decisions and create an environment to accelerate innovation & experimentation in your organisation

AWS Solutions Architects will facilitate a discussion and practical learning opportunity that will equip you with the knowledge you need to build a modern data platform on AWS and explore the capabilities that enable data visualisation, stream processing, machine learning and big data analytics

  • Learning Objectives
  • Target Audience

  • Building a Data Lake on AWS
  • Data Ingestion Strategies
  • Stream Processing and Big Data Analytics
  • Data Lake Consumption Patterns

Due to the practical nature of the workshop, it is expected attendees will have a entry level practical skillset on the AWS platform. free online training AWS Cloud Practitioner Essentials can be used to prepare.

For further questions, please contact your AWS account manager.

  • Solutions architects
  • System and application developers
  • Development managers
  • Data Engineers
  • Database administrators
  • IT Leaders

Event details

The modern data day will include presentations and labs therefore it will require the following:

  • A laptop with Wi-Fi running Microsoft Windows, Mac OS X, or Linux.
  • An Internet browser such as Chrome, Firefox, Safari, or Edge.
  • Internet access and port 22 access to the internet.
  • Basic hands on experience in databases, operating systems and applications

Lunch and catering will be provided throughout the day.

  • AGENDA 19th July

08:30 - 09:00 Registration & Check-in

Please arrive prior to 9am for registration and check-in.

09:00 - 09:30 Overview of the Data Lake

In this session, you will learn what makes data lakes different and the value proposition of a data lake.

09:30 - 10:15 Hydrating the Data Lake

In this session, we will look into the data ingestion patterns into the data lake and compare and contrast which patterns work best for which use cases.

10:15 - 10:30 Morning Tea

Morning tea will be provided by Amazon Web Services.

10:30 - 12:00 LAB: Hydrating the Data Lake

In this lab, you will use AWS data ingestion services to ingest data into your data lake.

12:00 - 12:45 Lunch

Lunch will be provided by Amazon Web Services.

12:45 - 1:30 Working within the Data Lake

In this session, we will dive deep into the optimisation strategies when storing and processing data from the data lake.

1:30 - 2:45 LAB: Transforming the Data Lake with AWS Glue

In this lab, you will get hands on experience on building serverless data transformations using AWS Glue.

2:45 - 3:00 Afternoon tea

Afternoon tea will be provided by Amazon Web Services.

3:00 - 3:30 Consuming the Data Lake

In this session, you will learn about the popular patterns for consuming data from the data lakes. We will also discuss the use cases that match best with each consumption pattern.

3:30 - 4:30 LAB [Option 1]: Stream processing with Kinesis

In this lab, you will hands-on experience with building a Kinesis streaming analytics application to apply real event-schedule-time fraud detection capbility.

3:30 - 4:30 LAB [Option 2]: Querying the Data Lake with Amazon Athena and Quicksight

In this lab, you will learn how to query the data lake with Amazon Athena and build a visual dashboard with Amazon Quicksight.

4:30 - 5:00 Overview of AWS Lake Formation

In this session, you will learn more about the newly announced data lake service AWS Lake Formation and how you can use it to build a data lake with AWS best practices.
